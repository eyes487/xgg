- [x] router
- [x] controller
- [x] service
- [x] model

# 运行方法
1. 克隆代码：

```
git clone https://github.com/eyes487/xgg.git
```

2. 进入项目根目录并安装依赖：

```
cd xgg
npm install / yarn
```


3. 运行项目

```
npm start

在浏览器中输入http://localhost:3000

```
